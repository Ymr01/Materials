## 📚 内容概览

本指南涵盖从C/C++基础到项目实战的完整知识体系，适合准备秋招的自动化/电子/计算机专业学生。

### 主要章节

- **语言基础**：C/C++深入、指针、内存管理
- **硬件基础**：数电模电、时序图、MCU架构
- **外设协议**：UART/SPI/I2C/CAN基础
- **RTOS**：FreeRTOS任务调度、队列、互斥锁
- **软件工程**：Git、Makefile、单元测试
- **传统嵌入式专项**：Bootloader、低功耗、看门狗
- **过来人建议**：时间管理、简历技巧、秋招策略、要不要读研
- **实战项目**：智能小车、温湿度监控系统

## 🎯 适用人群

- 有C语言基础
- 学过单片机（51/STM32）
- 了解基本电路知识
- 目标岗位：消费电子/工业控制/IoT/机器人方向的嵌入式开发

## 📅 学习周期

建议6-8个月，配合一段实习经历可显著提升竞争力。
