# 高三导数专题 · 系统梳理

> 适用于高考一轮/二轮复习，覆盖导数概念、计算、单调性、极值、恒成立、零点、不等式证明、隐零点、极值点偏移、综合压轴等 10 大核心模块。

## 📁 文件说明

- **`导数专题.html`** — 完整网页（69KB），包含：
  - 10 章系统知识点梳理
  - 30 道经典例题（每章 3 道），含详细步骤解答
  - 切线放缩不等式速查表
  - 考前冲刺清单

## 🚀 使用方法

### 在线访问
直接用浏览器打开 `导数专题.html`（双击或拖到浏览器窗口）。

**首次打开时**：
- 顶部会显示 "⏳ 数学公式加载中…"
- 等待 5-10 秒，公式渲染完成后红条自动消失
- **需要联网**（公式渲染依赖 MathJax CDN）

### 离线使用
如果需要完全离线（比如考前断网复习），可以：
1. 用浏览器打开后 **Ctrl+S 另存为完整网页**（会保存 MathJax 缓存）
2. 或者用 Chrome 的 "打印 → 另存为 PDF"（所有例题先点开展开）

## 🎯 核心亮点

### 必背工具（内置速查表）
- **6 个切线放缩不等式**（$e^x\geq x+1$、$\ln x\leq x-1$ 等）
- **4 种恒成立量词翻译**（任意/存在的正确对应）
- **3 类含参讨论模板**（一次/二次/判别式）
- **2 个零点充要条件**（单调性 + 存在性定理）

### 例题特色
- **30 道题覆盖历年高考热点**，每道都有难度（不是基础送分题）
- **详解折叠设计**：默认隐藏，点 "▶ 查看详解" 展开（适合先自己做）
- **关键启示 + 口诀**：每题最后总结方法论，而不是单纯解题

### 视觉设计
- **左侧固定导航**：章节可直接跳转
- **红框突出例题**，蓝/绿/金色分层标题，米黄底 + 红色系主题
- **移动端友好**（屏幕窄时自动堆叠）

## 💡 推荐用法

1. **知识回顾**：先看每章的理论梳理（前 3-5 屏）
2. **典型例题**：自己做一遍例题，再对照详解
3. **错题重做**：把不会的例题编号记下来，隔天再做
4. **考前速查**：
   - 最后一章 "🎯 考前冲刺清单" 快速扫盲点
   - 七、不等式证明章节的 "必背切线放缩" 表格打印出来贴桌角
